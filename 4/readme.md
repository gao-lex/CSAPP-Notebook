# 处理器体系结构

处理器必须执行一系列指令。一个处理器支持的指令和指令的字节编码成为其 **指令集体系结构(Instruction-Set Architecture,ISA)** 。`ISA` 在编译器编写者和处理器设计人员之间提供了一个概念抽象层，编译器你编写者只需要知道允许哪些指令，以及它们是如何编码的；处理器设计者必须建造出执行这些指令的处理器。
